Project Make It LLC makes web apps to provide tailored solutions for clients, and software-as-a-service for users. In making tailored solutions, we do design and IT work as needed.

We like web apps. A web app is just software that lives online, instead of trapped on one computer, so it can be used anywhere, any time, on any connected device. It also means everyone is up-to-date because the software only lives in one place, not on different computers running different versions with different data.
